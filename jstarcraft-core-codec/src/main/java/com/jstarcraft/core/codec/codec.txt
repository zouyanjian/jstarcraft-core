编解码框架,支持的编解码类型(CSV, Json, Kryo, ProtocolBufferX)

1.支持所有具有无参数构造器的类型,没有无参数构造器的类型无法注册到框架.
2.对象,枚举,数组,集合,映射都必须为具体类型,不能有通配符.
